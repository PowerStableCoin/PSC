Welcome to PSC

DAOPowerStableCoin (PSC) is a decentralized, algorithmic stable coin governed by a DAO. Using a burn-to-mint mechanism and robust oracles, PSC remains stable while adapting to market conditions.

Governance

Our DAO or combined Owner-DAO structure controls upgrades, parameter changes, asset approvals, etc. Proposals are subject to a delay (upgradeDelay) to allow community review.

If you hold governance tokens (or the designated role), you can propose changes on the Snapshot page (or an on-chain voting system).

Approved Assets

Below is an example list of approved stable coin collaterals:

  • USDT - 6 decimals
  • USDC - 6 decimals
  • DAI - 18 decimals

To add new assets, a proposal must pass, and the DAO (or Owner) calls setAsset.

Mint / Redeem PSC

Mint: Deposit an approved token (e.g., USDT), which is burned. PSC is minted using the aggregated price feed.

Redeem: If reserves are sufficient, PSC holders can convert stable coins back into underlying assets.

A DApp UI can be integrated here so users can connect their wallets and execute these functions directly.
